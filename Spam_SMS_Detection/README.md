Spam SMS Detection using Machine Learning

 Introduction

The Spam SMS Detection project uses Natural Language Processing (NLP) and Machine Learning to classify SMS messages as **Spam** or **Ham (Not Spam)**. It helps automatically filter unwanted messages and improves communication security.

 Objectives

* Classify SMS messages as Spam or Ham.
* Clean and preprocess text data.
* Extract features using NLP techniques.
* Train and compare different machine learning models.
* Evaluate model performance using standard metrics.

 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* NLTK
* Jupyter Notebook

 Project Workflow

1. Collected the SMS dataset.
2. Performed text cleaning and preprocessing.
3. Conducted Exploratory Data Analysis (EDA).
4. Converted text into numerical features using TF-IDF.
5. Trained multiple machine learning models.
6. Evaluated model performance.
7. Predicted whether new SMS messages are Spam or Ham.

 Machine Learning Models

* Multinomial Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)

Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix
* ROC-AUC Score

Future Improvements

* Perform hyperparameter tuning.
* Implement deep learning models such as LSTM.
* Explore BERT for advanced text classification.
* Develop a real-time spam detection system.
* Deploy the model using Flask or Streamlit.

Conclusion

This project demonstrates how NLP and machine learning can be used to detect spam SMS messages accurately. The developed model helps classify messages efficiently, improving message filtering, user experience, and communication security.
