Introduction

The Customer Churn Prediction project focuses on predicting whether a customer is likely to stop using a company's services. By analyzing customer information, account details, and service usage, machine learning models can help businesses identify customers who may leave and take steps to improve retention.

 Objectives
Predict customer churn using machine learning models.
Clean and preprocess the dataset.
Perform Exploratory Data Analysis (EDA).
Train and compare different classification algorithms.
Evaluate model performance using standard metrics.
